# Trabalho Prático 1: Algoritmo do Banqueiro

## Sobre o projeto

[cite_start]Este projeto implementa o Algoritmo do Banqueiro, com o objetivo de alocar recursos a múltiplos clientes (threads) garantindo que o sistema permaneça sempre em um estado seguro, prevenindo *deadlocks*[cite: 3, 6]. [cite_start]O projeto foi desenvolvido em **C**, utilizando a biblioteca **Pthreads** e *locks mutex* para evitar condições de corrida[cite: 35, 36].

## Estrutura de arquivos

- `src/main.c`: Código-fonte principal com a lógica do Banqueiro e das threads.
- [cite_start]`docs/`: Diretório contendo o relatório final em PDF[cite: 43].
